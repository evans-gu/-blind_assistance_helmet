# -blind_assistance_helmet
本作品以地瓜机器人RDK X5为主控，搭配STM32底板、导航模块、高清摄像头、音频接口与锂电电源，制成一体化智能助盲头盔。系统依托ROS 2架构，融合定位、视觉、语音、端侧大模型、小程序监控技术，服务视障人群出行。/This blind-assist helmet uses RDK X5 with STM32F1, GPS module, HD camera, audio port and lithium power. Based on ROS 2 nodes, it combines positioning, vision, voice, on-device LLM and Mini Program monitoring for visually impaired travelers.
